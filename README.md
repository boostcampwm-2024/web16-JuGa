### ❓ 이런 생각해보신 적 있나요?
- 주식을 한 번도 해보지 않았어요.
- 처음이라 시작하기가 두려워요.
- 자금이 부족해 다양한 시도를 못 해봤어요.
- 주식의 전반적인 시스템을 공부하고 싶어요.
- 친구와 함께 투자를 시작하고 싶어요.

### 🎯 Juga를 통해 이런 걸 경험하세요!
- 실제 주식 투자 전에 게임으로 먼저 경험해보세요.
- 리스크 없이 투자 감각을 키워보세요.
- 친구들과 함께 성장하는 재미를 느껴보세요.
- 경쟁하고 정보도 나누며 더 즐겁게 배워보세요.
